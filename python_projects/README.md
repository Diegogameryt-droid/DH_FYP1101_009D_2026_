# Proyectos Python

Esta carpeta contiene los proyectos desarrollados en Python.

## Estructura

- Cada proyecto debe tener su propia subcarpeta
- Incluir un `requirements.txt` con las dependencias
- Documentar el propósito y uso de cada proyecto

## Instalación de dependencias

```bash
pip install -r requirements.txt
```

## Ejemplos de proyectos

- Agregar subcarpetas con nombres descriptivos
- Mantener el código organizado y documentado
